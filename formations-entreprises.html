800 € / participant · Finançable OPCO / FAF

Faites travailler l'IA à votre place

3 modules indépendants au choix : Automatisation (n8n, workflows), Agents IA (n8n + API), Contenus visuels (Canva IA, Gemini, Adobe Firefly, Gamma). Vous choisissez le module qui correspond à vos priorités.

16h · 2 jours Visio live 3 à 16 participants Call suivi 30 min inclus
1 000 € / participant · Finançable OPCO / FAF

  • M2 Agents IA : n8n + appels API, agents autonomes
  • M3 Contenus visuels : Canva IA, Gemini, Firefly, Gamma, Copilot

Questions fréquentes

Ce que vous voulez savoir

Une session minimum de 3 participants — que se passe-t-il si on n'atteint pas ce seuil ?
Si une session n'atteint pas 3 inscrits 7 jours avant le démarrage, elle est reportée à la session suivante. Vous êtes prévenus immédiatement et votre inscription est transférée automatiquement.
Peut-on mixer TNS et salariés dans une même session inter ?
Oui. Le contenu est identique pour tous. Le financement se gère individuellement selon votre statut : FAF pour les TNS, OPCO pour les salariés.
Comment fonctionne le paiement pour les sessions inter ?
Paiement avant la session, par virement ou carte. Si vous financez via votre FAF, le dossier doit être déposé minimum 15 jours avant le début de la formation. On vous accompagne dans les démarches.
Quelle est la politique d'annulation ?
Annulation sans frais jusqu'à J-7. En dessous de 7 jours, la session est due. Un report vers une autre session est toujours possible sans frais, validé par Gabriel.
Quels outils informatiques faut-il avoir pour suivre la formation ?
Un ordinateur avec connexion internet. Tous les outils utilisés sont accessibles gratuitement ou en version d'essai. Aucune installation complexe, aucun prérequis technique pour le Niveau 1.

Une question avant de vous inscrire ?

15 minutes pour identifier le bon niveau, vérifier votre éligibilité au financement et choisir votre session. Gratuit, sans engagement.
